## 第2题

统计中所说的同质是指

A、对研究指标有影响的因素相同或相近  
B、对研究指标有影响的因素完全相同  
C、研究对象之间个体差异很小  
D、研究对象的测量指标变异很小  
E、以上都不对

你的答案：未记录  
正确答案：A

### 选项解析

- A 对：同质是指对研究指标有影响的主要因素相同或相近。
- B 错：同质不是要求完全相同，医学研究对象很少能做到所有因素完全一致。
- C 错：个体差异很小不等于同质；同质强调影响研究指标的主要因素。
- D 错：测量指标变异小是结果表现，不是同质的定义。
- E 错：A 已经正确。

记忆：同质不是“完全一样”，而是“主要影响因素相同或相近”。

## 第4题

抽样误差是指

A、个体指标值与参数值之差  
B、个体指标值与样本统计量值之差  
C、样本统计量值与总体参数之差  
D、个体指标值与个体指标值之差  
E、以上都对

你的答案：未记录  
正确答案：C

### 选项解析

- A 错：个体值与参数值的差异属于个体变异，不是抽样误差。
- B 错：个体值与样本统计量的差异也是个体观察值的离散，不是抽样误差。
- C 对：抽样误差是样本统计量与总体参数之间的差异。
- D 错：两个个体值之间的差异是个体差异。
- E 错：只有 C 符合抽样误差定义。

记忆：抽样误差 = 样本统计量 - 总体参数。

## 第7题

以下对概率描述错误的是

A、概率值界于0和1之间  
B、必然事件发生的概率为1  
C、概率用于描述某样本事件发生的强度  
D、样本含量足够大时，频率接近概率  
E、当随机事件发生的概率小于0.05或0.01时常称为小概率事件

你的答案：未记录  
正确答案：C

### 选项解析

- A 对：概率取值范围是 0 到 1。
- B 对：必然事件发生概率为 1。
- C 错：概率描述随机事件发生可能性的大小，不说“样本事件发生的强度”。
- D 对：样本量足够大时，频率会接近概率。
- E 对：P<0.05 或 P<0.01 常称为小概率事件。

记忆：概率说“可能性大小”，不是“强度”。

## 第8题

关于系统误差、测量误差和抽样误差下述正确的是

A、系统误差和测量误差不可避免  
B、系统误差和抽样误差不可避免  
C、测量误差和抽样误差不可避免  
D、三种误差都不可避免  
E、三种误差都可以避免

你的答案：未记录  
正确答案：C

### 选项解析

- A 错：系统误差可以通过严格设计和质量控制尽量避免或减少。
- B 错：系统误差不是不可避免；抽样误差通常不可避免。
- C 对：测量误差和抽样误差通常难以完全避免。
- D 错：系统误差不是“三种都不可避免”中的不可避免误差。
- E 错：测量误差和抽样误差通常不能完全避免。

记忆：系统误差要控制；抽样误差和测量误差通常难以完全避免。

## 第10题

统计工作的步骤为

A、研究设计、调查、整理资料、分析资料  
B、研究设计、试验、整理资料、统计推断  
C、研究设计、搜集资料、整理资料、分析资料  
D、研究设计、调查和试验、统计描述、统计推断  
E、研究设计、统计描述、统计推断、统计图表

你的答案：未记录  
正确答案：C

### 选项解析

- A 错：“调查”只是搜集资料的一种方式，统计工作步骤应概括为搜集资料。
- B 错：“试验”只是资料来源之一，且“统计推断”只是分析资料的一部分。
- C 对：统计工作基本步骤是研究设计、搜集资料、整理资料、分析资料。
- D 错：统计描述和统计推断属于统计分析内容，不是完整工作步骤。
- E 错：统计图表、统计描述、统计推断都属于分析或呈现环节，缺少搜集和整理资料。

记忆：统计工作步骤 = 设计、搜集、整理、分析。

## 本次错题压缩口诀

1. 同质：影响研究指标的主要因素相同或相近。
2. 抽样误差：样本统计量与总体参数之差。
3. 概率：随机事件发生可能性的大小。
4. 系统误差可控制，抽样误差难避免。
5. 统计工作步骤：设计、搜集、整理、分析。
