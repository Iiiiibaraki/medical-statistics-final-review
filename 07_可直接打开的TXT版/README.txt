## 维护规则

1. `.md` 是源文件。
2. `.docx` 是阅读版，统一放在 `06_可直接打开的Word版`。
3. `.txt` 是纯文本备份，统一放在 `07_可直接打开的TXT版`。
4. 每次计划、资料、进度或错题有变化，都要同步更新对应源文件，并重新生成 Word/TXT 版本。
5. 每次同步到 GitHub 前，都要更新 `README.md` 和 `00_总览与说明/14_文件结构总览.md`。
