## 文件夹说明

- `每日笔记/`：每日复习记录。
- `选择题按章节/`：按章节拆分的选择题。
- `选择题错题/`：错题整理；必须写原题、你的答案、正确答案，并解释每个选项为什么对或错。
- `SPSS截图/`：SPSS 输出截图，用于机考前快速复盘。
- `可直接打开的Word版/`：对应 Markdown 内容的 Word 阅读版。
- `课程网站资料/`：课程 PDF 与 SPSS `.sav` 练习数据。
