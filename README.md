# 医学统计学期末冲刺复习包

这是医学统计学期末复习资料仓库，包含复习计划、SPSS 操作速查、检验选择表、选择题题库、错题记录模板、课程资料索引和练习数据文件索引。

## 先打开

请先阅读：

- `README_先打开这个.md`
- `00_每日执行总表.md`
- `12_6月8日模拟考前加速计划.md`

## 主要内容

- `01_SPSS菜单与输出速查.md`：SPSS 菜单路径和输出表查看重点。
- `02_检验选择表.md`：根据题目类型选择统计检验。
- `03_SPSS操作题答题模板.md`：操作题结论写作模板。
- `04_错题本模板.md`：错题记录模板。
- `05_每日笔记模板.md`：每日复习笔记模板。
- `06_手录练习数据.md`：无数据文件时用于手录练习。
- `07_选择题按章节题库.md`：选择题汇总。
- `08_课程资料索引.md`：课程 PDF、教材内容和数据文件索引。
- `09_SPSS数据文件索引.md`：SPSS `.sav` 文件按章节索引。
- `10_SPSS试用期优先训练清单.md`：SPSS 试用期内优先训练安排。
- `每日笔记/`：每日复习记录。
- `选择题按章节/`：按章节拆分的选择题。
- `选择题错题/`：错题整理。
- `可直接打开的Word版/`：对应 Markdown 内容的 Word 版本。
- `课程网站资料待放入/`：课程 PDF 与 SPSS 练习数据。

## 使用建议

1. 每天从 `00_每日执行总表.md` 查看任务。
2. 做题前先用 `02_检验选择表.md` 判断统计方法。
3. 做 SPSS 操作题时对照 `01_SPSS菜单与输出速查.md` 和 `03_SPSS操作题答题模板.md`。
4. 每天把错题写入 `选择题错题/` 或按 `04_错题本模板.md` 新建记录。
