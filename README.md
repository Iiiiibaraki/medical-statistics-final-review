## 每日复习协作规则

当用户说“开始今天的学习”或类似表达后，按以下顺序执行：

1. 先生成当天的详细笔记，帮助用户建立本日主题的框架结构。
2. 详细笔记中尽量不要出现选择题原题，避免提前剧透，影响做题的新鲜感和自测效果。
3. 用户看完笔记后再开始做选择题。
4. 用户发来答案后，先批改，再整理错题；错题整理要解释错因和易混点。
5. 错题处理完成后，再进入 SPSS 实操练习。
6. 当天笔记、做题、错题和 SPSS 全部完成后，再同步内容到 GitHub。
7. 每次同步 GitHub 前，务必更新 `README.md` 的当前进度和相关说明。
8. 如果学习结束时间已经到凌晨，默认仍算作前一天学习的延续，不自动进入第二天内容，除非用户明确要求。

- `每日笔记/`：每日复习记录。
