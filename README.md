## 文件怎么用

建议打开顺序：

1. `README_先打开这个.md`
2. `13_时间线与文件命名说明.md`
3. `00_每日执行总表.md`
4. 当天的 `每日笔记/DayXX_日期_主题.md`
5. 做题或 SPSS 时再查对应速查表

## 主要文件

- `00_每日执行总表.md`：当前主计划，按日期执行。
- `01_SPSS菜单与输出速查.md`：SPSS 菜单路径和输出表查看重点。
- `02_检验选择表.md`：根据题目类型选择统计检验。
- `03_SPSS操作题答题模板.md`：操作题结论写作模板。
- `04_错题本模板.md`：错题记录模板，要求逐选项解析。
- `05_每日笔记模板.md`：每日复习笔记模板。
- `06_手录练习数据.md`：无数据文件时用于手录练习。
- `07_选择题按章节题库.md`：选择题汇总。
- `08_课程资料索引.md`：课程 PDF、教材内容和数据文件索引。
- `09_SPSS数据文件索引.md`：SPSS `.sav` 文件按章节索引。
- `10_SPSS试用期优先训练清单.md`：SPSS 试用期内优先训练安排。
- `11_每天开新对话复制这段.md`：每天新开对话时复制使用。
- `12_6月8日模拟考前加速计划.md`：模拟考前提速计划。
- `13_时间线与文件命名说明.md`：Day 编号、日期、文件命名总说明。

## 文件夹说明

- `每日笔记/`：每日复习记录。
- `选择题按章节/`：按章节拆分的选择题。
- `选择题错题/`：错题整理。错题必须写原题、你的答案、正确答案，并解释每个选项为什么对或错。
- `可直接打开的Word版/`：对应 Markdown 内容的 Word 版本，供直接打开阅读。
- `课程网站资料待放入/`：课程 PDF 与 SPSS 练习数据。

## 维护规则

1. 只保留 `.md` 和 `.docx`，不再维护 `.txt`。
2. `.md` 是编辑源文件。
3. `.docx` 是阅读版。
4. 修改 `.md` 后，同步更新对应 `.docx`。
5. 日期和 Day 编号冲突时，以 `13_时间线与文件命名说明.md` 和 `00_每日执行总表.md` 为准。
