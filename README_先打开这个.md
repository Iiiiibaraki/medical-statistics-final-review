## 每天打开顺序

1. 打开 `00_每日执行总表.md`，确认今天对应的 Day 和任务。
2. 打开当天的 `每日笔记/DayXX_日期_主题.md`。
3. 做选择题时打开 `选择题按章节/` 对应章节。
4. 做 SPSS 时打开 `01_SPSS菜单与输出速查.md`、`09_SPSS数据文件索引.md` 和 `10_SPSS试用期优先训练清单.md`。
5. 做错题后写入 `选择题错题/`，并逐个选项解释为什么对或错。
