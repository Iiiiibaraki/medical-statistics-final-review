## 2. 今天要建立的总体印象

医学统计学不是先背公式，而是先学会判断题目属于哪一类：

| 题目问什么 | 常见方法 |
|---|---|
| 描述一组身高、血压、血糖 | 描述统计 |
| 比较两组均数 | t检验 |
| 比较三组及以上均数 | 方差分析 |
| 比较率、构成比 | χ²检验 |
| 资料偏态或是等级资料 | 非参数检验 |
| 两个连续变量是否有关 | 相关分析 |
| 用多个因素解释一个连续结局 | 多元线性回归 |
| 用多个因素解释是否患病/是否有效 | Logistic回归 |

你后面每天学的本质就是：看到题目，判断它属于哪一行。

## 4. SPSS今天只要知道这些

SPSS有两个核心页面：

- 数据视图：像Excel一样录入数据，一行一个人，一列一个变量。
- 变量视图：设置每一列变量的名字、含义、分类编码。

变量设置最常见：

- Name：变量名，例如 `group`、`age`、`bp`。
- Label：变量解释，例如“分组”“年龄”“血压”。
- Values：分类变量编码，例如 `1=治疗组，2=对照组`。
- Measure：计量资料选 Scale，分类资料选 Nominal，等级资料选 Ordinal。

今天不用会跑分析，先把“数据是一行一个对象，一列一个变量”记住。
