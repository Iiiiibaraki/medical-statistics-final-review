## 第5题

变异系数

A、表示变量值的绝对离散度  
B、表示变量值的相对离散度  
C、表示均数的绝对离散度  
D、表示均数的相对离散度  
E、以上均不对

你的答案：D  
正确答案：B

### 选项解析

- A 错：绝对离散度常用标准差、全距、四分位数间距等表示，不是变异系数。
- B 对：变异系数 CV 表示变量值的相对离散程度。
- C 错：变异系数不是描述“均数的绝对离散度”。
- D 错：CV 的分母有均数，但它描述的是变量值相对于均数的离散程度，不是“均数本身的离散”。
- E 错：B 已经正确。

记忆：CV = 标准差 / 均数 × 100%，用于比较不同单位或均数差别较大资料的相对变异。

## 第9题

偏态分布宜用什么描述其分布的集中趋势

A、算术均数  
B、标准差  
C、中位数  
D、众数  
E、百分位数

你的答案：A  
正确答案：C

### 选项解析

- A 错：算术均数容易受极端值影响，不适合作为偏态分布的首选集中趋势指标。
- B 错：标准差描述离散程度，不描述集中趋势。
- C 对：偏态分布的集中趋势宜用中位数。
- D 错：众数可反映最常见值，但不是医学统计中描述偏态计量资料集中趋势的首选。
- E 错：百分位数可用于描述位置或参考值范围，但本题问集中趋势，最佳选中位数。

记忆：偏态资料：集中趋势用中位数，离散程度用四分位数间距。

## 第22题

正态分布中，什么相应的正态分布曲线的形状越扁平

A、μ越大  
B、μ越小  
C、σ越大  
D、σ越小  
E、以上都不对

你的答案：D  
正确答案：C

### 选项解析

- A 错：μ 决定曲线位置，μ 越大曲线整体向右移，不决定扁平程度。
- B 错：μ 越小曲线整体向左移，不决定形状扁平程度。
- C 对：σ 越大，数据越分散，正态曲线越扁平。
- D 错：σ 越小，数据越集中，曲线越高尖，不是越扁平。
- E 错：C 已经正确。

记忆：μ 管位置，σ 管形状；σ 越大越扁，σ 越小越尖。

## 第23题

标准正态分布曲线下，横轴上，从 -1.96 到均数的面积占总面积的百分比为

A、95%  
B、97.5%  
C、95.0%  
D、47.5%  
E、45%

你的答案：A  
正确答案：D

### 选项解析

- A 错：95% 是 -1.96 到 +1.96 之间的总面积，不是 -1.96 到均数的半边面积。
- B 错：97.5% 是 -∞ 到 +1.96 的左侧累计面积。
- C 错：和 A 一样，95.0% 对应中间对称区间 -1.96 到 +1.96。
- D 对：-1.96 到均数 0 是中间 95% 的一半，即 47.5%。
- E 错：45% 不是常用标准正态面积。

记忆：-1.96 到 +1.96 是 95%；均数到 1.96 是 47.5%。

## 第24题

标准正态分布曲线下，横轴上从 -2.58 到 1.96 间的面积占总面积的百分比为

A、5%  
B、90%  
C、95%  
D、97%  
E、99%

你的答案：C  
正确答案：D

### 选项解析

- A 错：5% 是 ±1.96 之外两侧尾部合计面积，不符合本题区间。
- B 错：90% 常对应约 ±1.64 的中间面积，不是本题区间。
- C 错：95% 对应 -1.96 到 +1.96，本题左端是 -2.58，右端是 +1.96，不是对称区间。
- D 对：-2.58 左侧约 0.5%，+1.96 左侧约 97.5%，二者相减约 97%。
- E 错：99% 对应 -2.58 到 +2.58 的中间面积，本题右端只有 +1.96。

记忆：遇到不对称区间，不能直接套 ±1.96 或 ±2.58，要用两端累计面积相减。

A、90%

- E 错：99% 约对应 -2.58 到 +2.58。

## 第30题

算术均数与中位数相比

A、抽样误差更大  
B、不易受极端值的影响  
C、更充分利用数据信息  
D、更适用于偏态分布资料  
E、更适用于有不确定数值的资料

你的答案：A  
正确答案：C

### 选项解析

- A 错：本题比较的是算术均数和中位数的特点，不能概括说算术均数抽样误差更大。
- B 错：不易受极端值影响的是中位数，不是算术均数。
- C 对：算术均数利用了每一个观察值的信息，因此比中位数更充分利用数据信息。
- D 错：偏态分布资料更适合用中位数描述集中趋势。
- E 错：有不确定数值的资料更适合用中位数，不是算术均数。

记忆：均数更充分利用信息，但怕极端值；中位数更稳健，适合偏态和不确定数值。

## 必补三类知识点

### 1. 变异系数 CV

`CV = 标准差 / 均数 × 100%`

- 用途：比较不同单位或均数差别较大资料的变异程度。
- 本质：变量值的相对离散程度。
- 特点：没有单位。

### 2. 偏态资料

- 集中趋势：中位数。
- 离散趋势：四分位数间距。
- 原因：偏态资料容易受极端值影响，均数和标准差不稳。

### 3. 标准正态面积

| 范围 | 面积 |
|---|---:|
| 均数左侧或右侧 | 50% |
| -1.96 到 +1.96 | 95% |
| -1.96 到均数 | 47.5% |
| +1.96 右侧 | 2.5% |
| -2.58 到 +2.58 | 99% |
| +2.58 右侧 | 0.5% |

## 今天必须背熟的 5 句话

1. 偏态资料：集中趋势用中位数，变异程度用四分位数间距。
2. 滴度资料：平均水平常用几何均数。
3. 比较不同单位资料的变异程度：用变异系数。
4. μ 管位置，σ 管形状；σ 越大，曲线越扁。
5. 正态分布面积：±1.96 是 95%，均数到 1.96 是 47.5%。
