## 第 1 题

原题：  
直线回归中，如果自变量 X 乘以一个不为 0 或 1 的常数，则

A、截距改变  
B、回归系数改变  
C、两者都改变  
D、两者都不改变  
E、以上情况都有

你的答案：C  
正确答案：B  
答题日期：2026-06-05  
对应 Day：Day07

### 错因

把“X 的计量尺度改变”误理解成整条回归方程的截距和斜率都会变。实际上，如果只是把自变量 X 乘以常数，相当于换了 X 的单位，原来的直线关系没有变，但“X 增加 1 个单位”的含义变了，所以回归系数 `b` 会改变。

设原方程为：

`Y = a + bX`

如果令 `X' = cX`，则 `X = X'/c`，代回去：

`Y = a + (b/c)X'`

所以截距仍为 `a`，回归系数变为 `b/c`。

### 选项解析

- A 错：截距不因单纯把 X 乘以常数而改变。
- B 对：X 的单位尺度改变，回归系数随之改变。
- C 错：不是两者都改变，截距不变。
- D 错：回归系数会改变。
- E 错：本题有明确规律，不是“以上情况都有”。

一句话记忆：

> X 换单位，斜率 b 换；截距 a 不动。

## 第 6 题

原题：  
关于直线相关系数和直线回归系数，错误的是

A、二者绝对值越大都表明变量间相关性越强  
B、二者的取值范围不同  
C、二者的意义不同  
D、对于同一资料，二者的符号相同  
E、二者的假设检验等价

你的答案：E  
正确答案：A  
答题日期：2026-06-05  
对应 Day：Day07

### 错因

把“相关系数 r”和“回归系数 b”的作用混在一起了。`|r|` 越大，说明线性相关越密切；但 `|b|` 越大，只说明回归线相对 X 轴更陡，不说明点更贴近回归线。

### 选项解析

- A 错：`|r|` 越大表示相关越密切；`|b|` 越大只表示斜率越陡，不能表示相关越密切。
- B 对：`r` 的范围是 `-1 <= r <= +1`，`b` 没有固定取值范围。
- C 对：`r` 表示线性关系方向和密切程度；`b` 表示 X 每增加 1 单位时 Y 平均改变多少。
- D 对：同一资料中，`r` 和 `b` 的符号相同。
- E 对：同一资料中，对 `r=0` 的检验和对 `b=0` 的检验等价。

一句话记忆：

> `r` 看贴不贴线，`b` 看线有多陡。

## 第 7 题

原题：  
如果两样本 r1=r2，n1>n2，那么

A、b1=b2  
B、b1>b2  
C、b1 和 r1 的假设检验的 t 值相等  
D、b1 和 r2 的假设检验的 t 值相等  
E、r1 和 r2 的假设检验的 t 值相等

你的答案：A  
正确答案：C  
答题日期：2026-06-05  
对应 Day：Day07

### 错因

看到 `r1=r2` 后，误以为回归系数 `b1=b2`。但 `b` 不只由 `r` 决定，还受 X 和 Y 标准差影响：

`b = r * Sy / Sx`

所以两个样本的 r 相同，不代表 b 一定相同。

### 选项解析

- A 错：`r1=r2` 不等于 `b1=b2`，因为 b 还受 `Sy/Sx` 影响。
- B 错：同理，不能根据 `r1=r2` 和 `n1>n2` 推出 `b1>b2`。
- C 对：同一份资料中，检验回归系数 `b=0` 和检验相关系数 `r=0` 的 t 值相等。
- D 错：`b1` 属于第 1 个样本，`r2` 属于第 2 个样本，不能跨样本配对说 t 值相等。
- E 错：即使 `r1=r2`，因为 `n1>n2`，相关系数检验的 t 值也不一定相等。

一句话记忆：

> 同一资料里 r 和 b 检验等价；不同资料之间不能只看 r 相等就推出 b 相等。

## 第 8 题

原题：  
直线回归中 X 与 Y 的标准差相等时，则

A、b=a  
B、b=r  
C、b=1  
D、r=1  
E、r=0

你的答案：C  
正确答案：B  
答题日期：2026-06-05  
对应 Day：Day07

### 错因

把“X 与 Y 的标准差相等”误推成 `b=1`。实际上回归系数和相关系数的关系是：

`b = r * Sy / Sx`

当 X 与 Y 的标准差相等时，`Sy/Sx = 1`，所以：

`b = r`

这不代表 `b=1`，除非同时 `r=1`。

### 选项解析

- A 错：`b` 是回归系数，`a` 是截距，标准差相等不能推出二者相等。
- B 对：由 `b = r * Sy / Sx`，当 `Sy = Sx` 时，`b = r`。
- C 错：只有在 `r=1` 且 `Sy=Sx` 时，才有 `b=1`。
- D 错：标准差相等不能推出完全相关。
- E 错：标准差相等也不能推出无线性相关。

一句话记忆：

> `b = r * Sy / Sx`；标准差相等时，`b = r`，不是 `b = 1`。

复刷完成：[ ]

## 错因分类

- 概念混淆：把 `r` 和 `b` 的意义混在一起。
- 公式不熟：`b = r * Sy / Sx` 没有形成自动反应。
- 条件误推：把 `r1=r2` 误推为 `b1=b2`，把 `Sx=Sy` 误推为 `b=1`。
- 单位意识不足：没有立刻想到 X 换单位会影响斜率 b。

## 高频易混点

| 易混点 | 正确记法 |
|---|---|
| `|r|` 越大 | 线性相关越密切 |
| `|b|` 越大 | 回归线越陡 |
| r 的范围 | `-1 <= r <= +1` |
| b 的范围 | 没有固定范围 |
| 同一资料中 r 和 b | 符号相同，假设检验等价 |
| 不同资料中 r 相等 | 不能推出 b 相等 |
| b 与 r 的关系 | `b = r * Sy / Sx` |
| `Sx = Sy` | `b = r` |
