## 第 3 题

原题：  
比较某地在两个年份几种传染病的发病率可用

A、构成比条图  
B、复式条图  
C、线图  
D、直方图  
E、圆图

你的答案：A  
正确答案：B  
答题日期：2026-06-08  
对应 Day：Day08

### 错因

把“几种传染病发病率的比较”看成了“构成比”。本题不是问这些疾病占总数的组成比例，而是比较不同疾病、不同年份的发病率高低。两个年份加上几种疾病，属于多组分类指标比较，适合用复式条图。

### 选项解析

- A 错：构成比条图用于表达整体由哪些部分组成，本题是发病率比较。
- B 对：复式条图适合同时比较两个年份中几种传染病的发病率。
- C 错：线图主要看随时间变化趋势；本题只有两个年份和多个类别，更适合复式条图。
- D 错：直方图用于连续变量频数分布。
- E 错：圆图用于构成比。

一句话记忆：

> “两个年份 + 几个类别 + 比较率高低”优先想到复式条图。

## 第 11 题

原题：  
下列哪种统计图纵坐标必须从0开始

A、半对数线图  
B、散点图  
C、条图  
D、线图  
E、百分条图

你的答案：B  
正确答案：C  
答题日期：2026-06-08  
对应 Day：Day08

### 错因

把“显示关系”的散点图和“比较大小”的条图规则混在一起了。条图用条的高度表示数量或率，如果纵轴不从 0 开始，容易夸大类别之间的差异，因此条图纵坐标通常必须从 0 开始。

### 选项解析

- A 错：半对数线图的纵轴是对数尺度，没有 0 点。
- B 错：散点图主要看两个计量变量之间的关系，纵轴不要求必须从 0 开始。
- C 对：条图通过条高比较大小，纵坐标必须从 0 开始。
- D 错：普通线图主要看趋势，纵轴不一定必须从 0 开始。
- E 错：百分条图强调构成比，不是本题最标准答案。

一句话记忆：

> 问“纵轴必须从 0 开始”，先想条图。

## 第 13 题

原题：  
关于半对数线图，哪项是错误的

A、纵轴为对数尺度，横轴为算术尺度  
B、纵坐标没有零点  
C、通过绝对差值而不是相对比来反映事物发展速度  
D、纵坐标各单元等距，但同一单元内不等距  
E、当事物数量相差悬殊时，比普通线图更适宜比较事物的发展速度

你的答案：B  
正确答案：C  
答题日期：2026-06-08  
对应 Day：Day08

### 错因

把“纵坐标没有零点”误判成错误。半对数线图的纵轴是对数尺度，对数坐标不能有 0 点，所以 B 是正确说法。真正错误的是 C：半对数线图主要比较相对发展速度，不是比较绝对差值。

### 选项解析

- A 对：半对数线图通常横轴为算术尺度，纵轴为对数尺度。
- B 对：对数坐标没有 0 点。
- C 错：半对数线图反映的是相对变化速度，不是绝对差值。
- D 对：对数坐标各大单元等距，但同一单元内刻度不等距。
- E 对：数量相差悬殊时，用半对数线图更适合比较发展速度。

一句话记忆：

> 半对数线图看相对速度；纵轴没有 0 点是它的正常特点。

## 第 15 题

原题：  
多元线性回归分析中说明应变量与自变量间相关关系密切程度的指标为

A、复相关系数  
B、决定系数  
C、标准化回归系数  
D、回归系数  
E、剩余标准差

你的答案：C  
正确答案：A  
答题日期：2026-06-08  
对应 Day：Day08

### 错因

把“比较多个自变量相对作用大小”的标准化回归系数 Beta，误用于判断模型相关关系密切程度。多元线性回归里，说明因变量与多个自变量整体相关关系密切程度的是复相关系数 R。

### 选项解析

- A 对：复相关系数 R 说明因变量与多个自变量整体相关关系的密切程度。
- B 错：决定系数 R Square 说明模型解释因变量总变异的比例。
- C 错：标准化回归系数 Beta 用于比较不同单位自变量的相对影响大小。
- D 错：回归系数说明某个自变量对因变量的影响方向和大小。
- E 错：剩余标准差反映残差离散程度，不是相关密切程度。

一句话记忆：

> 密切程度看 R，解释百分比看 R Square，相对作用大小看 Beta。

## 错因分类

- 图形关键词误读：把“发病率比较”误当成“构成比”。
- 坐标轴规则混淆：散点图看关系，条图才强调纵轴从 0 开始。
- 半对数线图核心没抓牢：它比较相对发展速度，纵轴没有 0 点。
- 多元回归指标混淆：R、R Square、Beta 的功能没有完全分开。

## 高频易混点

| 易混点 | 正确记法 |
|---|---|
| 构成比 | 圆图、百分条图 |
| 几个类别之间比较率或数量 | 条图、复式条图 |
| 条图纵坐标 | 必须从 0 开始 |
| 散点图 | 看两个计量变量之间关系 |
| 半对数线图 | 比较相对发展速度 |
| 半对数线图纵轴 | 对数尺度，没有 0 点 |
| 复相关系数 R | 相关关系密切程度 |
| 决定系数 R Square | 解释因变量变异百分比 |
| 标准化回归系数 Beta | 比较不同自变量相对作用大小 |
